Personal Portfolio Website - Vivek Netha

This is a modern, responsive, and minimalist personal portfolio website
built to showcase my identity, technical skills, and educational background.

Key Features
Responsive Design: 
The website is optimized for all screen sizes (mobile, tablet, and desktop) using CSS Flexbox and media queries.  

Dark/Light Mode Toggle: 
Integrated a theme switcher that allows users to toggle between light and dark modes with a single click.  

Interactive UI: 
Includes a custom-built hamburger menu for mobile navigation and smooth scrolling across sections.  

Expertise Section: 
Displays core skills using clean, animated cards with icons.  

Contact Portal: 
A functional-style contact form integrated with social media links for professional networking.  

Technical Stack
HTML5: For semantic structure.
CSS3: Custom styling, including CSS Variables for the theme-switching logic.  
JavaScript (ES6+): Handles the mobile menu toggle, smooth scrolling, and dark mode state management.  
Google Fonts: Utilizes fonts like Poppins, Montserrat, and Google Sans.  

Project Structure
index.html: Main portfolio structure and JS logic.  
style.css: Core styling and responsive media queries.  
me.png: Profile photograph.  


How to Run
Clone or download this repository.
Ensure index.html, style.css, and your image (me.png) are in the same folder. 
Open index.html in any modern web browser.

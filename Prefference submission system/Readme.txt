NSS Preference Submission Portal - IIT Bombay
This is a formal preference submission web application designed for NSS IIT Bombay. It allows students to submit their department preferences and stores the data locally within the browser.

Key Features
Official Branding: Features the NSS logo and the official motto "Think we not me" within a custom navbar.  
Comprehensive Form: Collects user details including Name, Roll Number, LDAP Email, Personal Email, and Contact Number.  
Preference Selection: Allows users to rank their top 3 department choices from official NSS wings like Educational Outreach and Green Campus.  
Data Persistence: Uses browser localStorage to ensure submitted data remains available even after a page refresh.  
Real-time Display: Submissions are displayed immediately in a structured table format below the form.  
Management Tools: Includes functionality to edit or delete previously submitted entries to maintain data accuracy.

Technical Stack
HTML5: Used for the portal structure and form elements.  
CSS3: Custom styling for a clean, professional IITB-themed layout with responsive media queries.  
JavaScript (ES6+): Handles form validation, localStorage operations (setItem/getItem), and dynamic table rendering.  

Project Structure
q2.html: The main portal file containing the branding, form, and submission table.  
q2.css: Stylesheet for the NSS theme and responsive layout.  
q2.js: Logic for saving, loading, and managing submissions in local memory.  

How to Use
Open index.html in any modern web browser.
Enter your personal and contact details in the provided fields.  
Select your 1st, 2nd, and 3rd department preferences from the dropdown menus.  
Click "Submit Preferences" to save your data.  
View your entry in the "Previous Submissions" table.
